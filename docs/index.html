What you get

Built like a Mac app, not a browser tab.

🔒

Isolated sessions

Each account gets its own persistent data store. Your numbers never mix — log in once, stay logged in.

🖼️

Real identity

The sidebar shows each account's actual profile photo and number, detected automatically. Rename or recolor anytime.

🔔

Notifications that work

Sender and message preview, delivered by macOS. Click one to jump straight to that account.

🍃

Memory you control

Per-account modes let idle numbers sleep to almost zero RAM and wake to check on a timer.

⌨️

Keyboard-first

⌘1…⌘9 to switch, ⇧⌘] / ⇧⌘[ to cycle, ⌘R to reload. A Dock badge counts every unread.

📞

Calls included

Camera and microphone are granted for WhatsApp voice and video calls, right inside the window.

Memory modes

Decide what each number is allowed to cost.

Set it per account. The default keeps one active and lets the rest sleep.

⚡ Always on
RAM while idle

Connected the whole time. Notifications land instantly, with content. Costs the most memory.

🍃 Economy DEFAULT
RAM while asleep

Sleeps to near zero and wakes every few minutes to check. You get a heads-up notification, slightly delayed.

✋ Manual
RAM while closed

Off until you open it. Zero memory, no background notifications. For numbers you only check now and then.

🛡️

Nothing leaves your Mac.

MultiZap is a local wrapper around the official WhatsApp Web — no server, no analytics, nothing sent anywhere. Sessions and cached photos live only on your machine. It wraps WhatsApp Web as-is, so there's no ban risk beyond using WhatsApp Web normally.
